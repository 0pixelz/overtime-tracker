# overtime-tracker

A self-contained web app for tracking weekly overtime hours.

**Live app:** https://0pixelz.github.io/overtime-tracker/

## Features
- Track arrival/departure times and lunch breaks for each workday
- Automatic overtime/deficit calculation
- Day-off management (paid leave, absences)
- Task notes per day
- Weekly history and calendar view
- Data persists locally in the browser (localStorage)

## Usage
Visit the live app at the link above. All data is stored in your browser — nothing is sent to a server.
